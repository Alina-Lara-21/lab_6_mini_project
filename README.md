# Calculator
Calculator for simple arithmetic operations

## Features

- **Basic Arithmetic**: Add, Sub, Mult, Divide 

- **Interactive CLI**: Can be exicuted using command-line interface
- **Error Handling**: Validation for invalid inputs and division by zero

## Project Structure

~~~~
calculator/
> main.py          # user interface interactions
>calculator.py    # calculator logic
>README.md        # Documentation for the project
>requirements.txt #Python dependents 
~~~~

## Usage

Run the calculator from the command line:

```bash
python main.py
```


## Operating Instructions

1. Run `main.py`
2. Select an operation by using (1-7)
3. Enter numbers you want to compute when prompted
4. View the calculation result
5. Keep going or select 7 to exit

## Keyboard Shortcuts (Web Calculator)

- `0-9`: enter numbers
- `+`, `-`, `*`, `/`: operators
- `.`: decimal point
- `Enter` or `=`: calculate result
- `Backspace`: delete last character
- `Escape`: clear all
- `%`: percent
- `P`: pi constant
- `E`: e constant
- `S`, `C`, `T`, `R`, `L`, `N`: sin, cos, tan, sqrt, log, ln
